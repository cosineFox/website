# Canvas Labworks

Personal site and photography portfolio. SvelteKit + Svelte 5.

## Pages

- `/main` - about me, Bluesky feed, Steam games
- `/imageworks` - photography gallery (pulls from Bluesky)
- `/maker` - projects
- `/maker/dashboard` - weather dashboard

## Run locally

```bash
npm install
npm run dev
```
